## üìö Using Laddr

Once Laddr is running, you can start creating your agents and tasks. Here are some basic commands you can use:

- **Create an Agent:** Use the command `create_agent(name)` to create a new agent.
- **Assign a Task:** Use the command `assign_task(agent_name, task)` to assign a task to the created agent.
- **Monitor Agents:** Use the command `monitor_agents()` to see the status of all active agents.

Check the Laddr documentation for more detailed information on how to set up and use agents effectively.
